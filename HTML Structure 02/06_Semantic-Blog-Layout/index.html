New feature:JPEG XL image format

JPEG XL image format is modern image format optimized for web environments. JPEG XL generally has better compression than WebP,JPEG,PNG and GIF and is designed to supersede them.

Learn more at: https://en.wikipedia.org/wiki/JPEG_XL

New feature: Cookie Store API

Cookie Store API is a JS API for readimg and modifying cookies. Compared to the existing 'document cookie' method, the API provides a much more modern interface, which can also be used in service workers.

Learn more at: https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/API/cookies/CookieStore
